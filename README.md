# Aleph

A Simple Chess Engine


## Features

### Search
- PV Search
- Transposition Table
- Check Extensions
  
### Move Ordering
- MVV-LVA
- Killer Move Heuristics
- History Heuristics
- PV Table Lookup
- Transposition Table Lookup
  
### Evaluation
- Piece Value
- Piece Square Tables
- Piece Activity
- Basic Pawn Structure
- Basic King Protection
